暗黑模式切换

由于本网站使用了 element-plus 组件库，所以在element-plus官方暗黑模式下增加vitepress的暗黑模式样式

暗黑模式 
<template>
  <div>
   <span>暗黑模式</span> <el-switch v-model="mode" @change="onChange" />
  </div>
</template>
<script setup>
import { reactive,ref } from 'vue';
const mode = ref(false)

function onChange(val) {
  if(val){
    // 使用了 element-plus 的暗黑模式
    document.documentElement.classList.add('dark')
  }else{
    document.documentElement.classList.remove('dark')
  }
}

</script>

给网站简单的添加暗黑模式方式

/* filter: invert() — 是从0到1的刻度，1是从白变黑 */
/* filter: hue-rotate() — 改变你元素的颜色，同时尽量保持原本相同的色系，取值0deg到360deg。 */

html[theme='dark-mode'] {
  filter: invert(1) hue-rotate(180deg);
}
/* 要注意把图片的色调转换回来 */
html[theme='dark-mode'] img{
  filter: invert(1) hue-rotate(180deg);
}

vitepress 0.22.4 增加暗黑模式

在element plus 组件库下增加暗黑模式

  • 新增自定义主题样式文件 docs/.vitepress/layout/index.css, 增加以下样式
html {
  /* 过渡 */
  transition: all 0.3s ease-in-out;
}

:root.dark {
  filter: invert(1) hue-rotate(180deg);
}

/* 解决图片色调被全局暗黑模式转换了 */
:root.dark img {
  filter: invert(1) hue-rotate(180deg);
}
