JS数据类型

基本数据类型

  • Boolean: 布尔类型

  • Null

  • Undefined

  • String: 字符串类型

  • Number: 数字类型

    • NaN 也属于数字类型。但是NaN不属于自身。

  • BigInt

    • 可以表示任意精度的整数。
    • 通过在整数末尾附加字母 n 或调用构造函数来创建的。
    • 可以安全存储和操作大整数，包括超过安全整数Number.MAX_SAFE_INTEGER限制。

  • Symbol: 是唯一且不可修改的值。

      // 唯一的值
  const sym1 = Symbol()
  const sym2 = Symbol()

  console.log(sym1 === sym2); // false

  • 对象类型是引用类型,指内存中的可以被标识符引用的一块区域。
  • 对象拥有 数据类型 和 访问器属性(get和set)。
