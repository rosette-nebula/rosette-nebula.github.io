github的workflow

自动发布版本与自动更新版本日志

文档

  • 创建.github/workflow/release.yml，并添加以下配置
on:
  push:
    branches:
      - main
name: release-please
jobs:
  release-please:
    runs-on: ubuntu-latest
    steps:
      - uses: google-github-actions/release-please-action@v3
        with:
          release-type: node
          package-name: release-please-action
