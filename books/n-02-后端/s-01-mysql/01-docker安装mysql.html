docker 安装mysql5.7

  • 安装
docker pull mysql:5.7
1
  • 创建 mysql 容器
docker run -p 3306:3306 --name mysql -v /www/mysql/conf:/etc/mysql/conf.d -v /www/mysql/logs:/logs -v /www/mysql/data:/var/lib/mysql -e MYSQL_ROOT_PASSWORD=123456 -d mysql:5.7
1

INFO

命令说明：

  1. -p 3306:3306：将主机的3306端口映射到docker容器的3306端口。
  2. --name mysql：运行服务名字
  3. -v /www/mysql/conf:/etc/mysql/conf.d ：将主机/www/mysql录下的conf/my.cnf 挂载到容器的 /etc/mysql/conf.d
  4. -v /www/mysql/logs:/logs：将主机/www/mysql目录下的 logs 目录挂载到容器的 /logs。
  5. -v /www/mysql/data:/var/lib/mysql ：将主机/www/mysql目录下的data目录挂载到容器的 /var/lib/mysql
  6. -e MYSQL_ROOT_PASSWORD=123456：初始化 root 用户的密码。
  7. -d mysql:5.7 : 后台程序运行mysql5.7
  • 查看容器
docker ps
